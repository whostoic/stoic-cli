stoic version       : Prints currently installed version.
stoic uninstall     : Removes stoic-cli from you machine.
stoic update        : Updates stoic-cli to latest version.

cdb <number>        : Change directory backward <number> times.
                      <number> argument is optional. If no argument is provided, it will change one directory backwards.
gr <directory>      : Go to <directory> from git root directory.
                      <directory> argument is optional. If no argument is provided, it will set the git root directory as pwd.
shc <key>           : Got to predefined path. Accepts exactly one argument as <key> and reads path from .stoic-shortcuts.
mkcd <directory>    : Creates a directory with <directory> name and sets it as pwd.
